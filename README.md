## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

clone repo: `git clone https://github.com/Yerimah/Contact-form`

then
`cd Contact-form`

To run locally
`npm run start`

### Install

run `npm install` to install dependencies
